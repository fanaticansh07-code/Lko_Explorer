# 🏰 Lko-Explorer: A Curated Local Tourism Guide

**Lko-Explorer** is a responsive web application designed to showcase the heritage, culinary delights, and hidden gems of Lucknow. This project was developed as a Minor Project for the B.Sc. Computer Science program at Lucknow University.

---

## 🚀 Features
- **Dynamic Content:** Uses JavaScript to inject data into the UI without hardcoding HTML.
- **Nawabi Aesthetic:** A custom UI theme using Royal Blue and Gold accents.
- **Category Filtering:** Users can toggle between Heritage, Food, and Hidden Gems.
- **Fully Responsive:** Optimized for Laptops, Tablets, and Smartphones using Bootstrap 5.
- **Google Maps Integration:** Direct links to every location for easy navigation.

## 🛠️ Technology Stack
- **Frontend:** HTML5, CSS3 (Flexbox/Grid), JavaScript (ES6+).
- **Framework:** Bootstrap 5.0 (CDN).
- **Typography:** Google Fonts (Playfair Display & Poppins).
- **Icons:** Bootstrap Icons.
